# EPIC 在线服务开发者协议

最后更新日期： 2024 年 2 月 15 日

Epic 开发了一套在线服务，供您在视频游戏中使用。Epic 有意在向您提供这些服务，供您在视频游戏中使用，其方式和质量水平与 Epic 对其自身视频游戏的期望相同。

Epic 提供两层服务：实时服务和预览服务。实时服务是 Epic 计划长期提供的服务。预览服务是早期的预览版本，随着 Epic 继续开发，预计会发生改变，且 Epic 会对预览服务征求您的反馈意见。除非相关服务附录中另有规定，否则所有服务均为实时服务。

请仔细阅读本协议。本协议是一份法律文件，说明了与您使用 SDK 和相关服务有关的权利和义务。只有遵守本协议条款，您才能使用这些服务。单击“接受”，下载或使用 SDK，或访问服务，即表示您同意受本协议条款的约束。如果您不同意或不能同意本协议的条款，请勿下载或使用 SDK 或访问任何服务。

在本协议中使用的某些词汇或短语的定义应以其在本协议中的特定含义为准。该等词汇和短语的定义请见下文第 12 条。

如果您的主要居住地（如果您是公司或学术机构等法人实体的情况下，则为主要营业地）在美国，则与您签订协议的主体是 Epic Games, Inc.。如果您的主要居住地（或主要营业地）不在美国，则与您签订协议的主体是 Epic Games Commerce GmbH。

您确认您已年满 13 岁，或已年满您所居住国家/地区的数字成人年龄（按较高的年龄来算）。

- ## 1\. 协议目的
本协议旨在规定有关您使用 SDK 和相关服务的条款。接受本协议后，您可以访问以下标准服务的服务附录中列出的服务。您可以接受适用的服务附录，访问更多的服务；这些服务附录可能包括服务或功能特定条款。本协议连同任何服务附录和任何通过引用并入的文件构成一份协议。

- ## 2\. 服务
  
    - 2\.1\. 在本协议期限内，Epic 将提供服务，并尽商业上合理的努力，以与 Epic 为其自身游戏提供的服务基本相同的方式和质量水平提供此类服务，同时考虑此类服务是预览服务还是实时服务。
  
    - 2\.2\. Epic 在第 2.1 条下的义务取决于以下条件：(a) 您持续、善意地配合执行所有必要的或 Epic 合理要求的任务，整合 SDK 并确保您的视频游戏与 SDK 和服务兼容，以及 (b) 您遵守本协议的所有适用条款。
  
    - 2\.3\. 如果您在一年或更长时间内未使用或访问特定服务，Epic 可在提前三十 (30) 天通知后，删除通过该服务存储或维护的与此类服务相关的任何数据。

- ## 5\. 更新和修订
  
    - 5\.1\. Epic 可能会不时发布 SDK 更新，并通过开发者门户提供此类更新。Epic 会尽商业上合理的努力，确保每项 SDK 更新都与集成了 SDK 先前版本的视频游戏二进制兼容。预计您会在 SDK 更新发布后三 (3) 年内使用 SDK 更新来更新您的视频游戏。在此期限之后，Epic 会尽商业上合理的努力来确保任何实时服务的持续向后兼容性，但是 Epic 没有任何义务在收到 SDK 更新超过三 (3) 年以后支持任何先前版本。
  
    - 5\.2\. Epic 可能会在提前三 (3) 年向您发出书面通知后停止支持任何实时服务。
  
    - 5\.3\. 在事先书面通知您的情况下，Epic 可能会随时对任何预览服务进行重大更改。
  
    - 5\.4\. Epic 可以通过向您发送通知、在您登录开发者门户或下载 SDK 更新时向您提供修订版协议的数字访问权等方式，自行决定随时发布修订版协议。您并非必须接受修订版协议。但是，为了使用新的 SDK 更新或访问 Epic 提供的新服务，您必须接受修订版协议。尽管有上述规定，Epic 仍可针对 SDK 的当前版本或当前服务修订本协议，以符合法律或监管要求；在这种情况下，Epic 会向您发送合理通知。
